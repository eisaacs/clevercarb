# clevercarb
Mountaintop Project

## What you'll need
- Have Git installed
  - [Use this link](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- Have Node.js & NPM installed
  - [Use this link](https://nodejs.org/en/download/)

## Running locally
- Clone this repo
  - `$ git clone https://github.com/elklein96/clevercarb`
- Go into the directory
  - `$ cd clevercarb`
- Install dependencies  
  - `$ npm install && bower install`
  - Note, install `bower` using `$ npm install -g bower`
- Install Grunt if you don't already have it 
  - `$ npm install -g grunt-cli`
- Run the app
  - `$ grunt`
- Go to [http://localhost:3000](http://localhost:3000) in your browser
